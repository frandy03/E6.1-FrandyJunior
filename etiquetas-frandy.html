Etiquetas para trabajar con texto

  • <h1>, <h2>, <h3>, <h4>, <h5> y <h6>

    • Estas etiquetas se utilizan para definir titulos.
    Ejemplo:

    Encabezado 1

    Encabezado 2

    Encabezado 3

    Encabezado 4

    Encabezado 5
    Encabezado 6

  • <p>

    • Se utiliza para definir parrafos.

  • <b> o

    • Se utiliza para poner un texto en negrita.
    Ejemplo:
    Texto en negrita

  • <u>

    • Se utiliza para poner un texto subrayado.
    Ejemplo:
    Texto subrayado

  • <i>

    • Sirve para poner un texto en cursiva.
    Ejemplo:
    Texto en cursiva

  • <small>

    • Esta etiqueta define un texto pequeño.
    Ejemplo:
    <small> Texto </small>
    Texto pequeño

  • <br>

    • Esta etiqueta sirve para hacer un salto de linea.

  • <sub>

    • Esta etiqueta sirve para poner un subindice.
    Ejemplo:
    <sub> </sub>
    Esto es un Subindice

  • <sup>

    • Esta etiqueta sirve para poner un superindice.
    Ejemplo:
    <sup> </sup>
    Esto es unSuperindice

  • < >

Etiquetas para imagenes

  • <img>

    • Sirve para definir imagenes en una pagina HTML5 en formatos JPEG, PNG,SVG o GIF.
    Ejemplo:
    <img src="imagen.png">

Etiquetas para enlaces

  • <a>

    • Sirve para definir un enlace.
    Ejemplo:
    <a href= "enlace"></a>
    Mi perfil en GitHub

Etiquetas para citas

  • <cite>

    • Esta etiqueta marca una referencia a una fuente de un texto citado.
    Ejemplo:
    <cite> </cite>

Etiquetas para listas

  • <ul>

    • Esta etiqueta define listas de articulos no ordenadas.
    Ejemplo:

    • Primero
    • Segundo
    • Tercero

  • <ol>

    • Esta define listas de articulos en forma ordenada.
    Ejemplo:

    1. primero
    2. Segundo
    3. Tercero
